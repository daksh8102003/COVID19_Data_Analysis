# 🦠 COVID-19 Data Analysis Project (Beginner)

This beginner-friendly data analysis project uses global COVID-19 data to demonstrate basic data cleaning, exploration, and visualization techniques using Python.

## 📊 Objective

To explore trends in COVID-19 cases over time and across countries using real-world data.

---

## 📁 Dataset

- Source: [Our World in Data](https://ourworldindata.org/covid-deaths)
- File: `owid-covid-data.csv`  
- Sample includes data for India over 3 days (can be expanded to full dataset).

---







## 🛠️ Tools & Technologies

- Python 3.x
- Jupyter Notebook (via Anaconda)
- Libraries:
  - `pandas`
  - `matplotlib`
  - `seaborn`

Install dependencies using:
```bash
pip install -r requirements.txt


----




📌 Features
Load and inspect a real-world COVID-19 dataset.

Clean and filter data for specific countries.

Visualize daily new cases using line charts.

Compare total cases using bar charts.
