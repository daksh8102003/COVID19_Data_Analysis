# COVID-19 Data Analysis Project

This is a beginner-level data analysis project using COVID-19 data.

## Files Included:
- `owid-covid-data.csv`: Sample COVID-19 dataset
- `COVID19_Analysis.ipynb`: Jupyter Notebook for analysis
- `README.txt`: This file

## How to Run:
1. Open Anaconda Navigator
2. Launch Jupyter Notebook
3. Navigate to the project folder and open `COVID19_Analysis.ipynb`
4. Run the cells to begin analysis
